# ETHOS

## 1. Steps to Run:

### i. Clone the Repository:
- ```bash
  git clone https://github.com/ZeroThreshold/ethos.git
  ```

### ii. Into the Folder:
- ```bash
  cd ethos
  ```

### iii. Installing Packages:
- ```bash
  npm install
  ```

### iv. Running ORM:
- ```bash
  npx prisma generate
  ```

### v. App Run
- ```bash
  npm run dev
  ```
